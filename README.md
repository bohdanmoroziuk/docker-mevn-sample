# Docker MEVN Sample

> Example of Dockerizing MEVN Application 

## Project structure

```txt
.
├── backend
│   ├── Dockerfile
│   └── ...
├── frontend
│   ├── Dockerfile
│   └── ...
├── compose.yaml
└── ...
```

## Available scripts

Start the development container:

```bash
make start
```

Stop the development container:

```bash
make stop
```
